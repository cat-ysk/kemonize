kemonize.rb
====

BrainfuckのソースコードをKemonoへ変換するコンバータだよ！

# つかいかた
```bash
$ ./kemonize.rb helloworld.bf > helloworld.kemono
```

# しゅつりょく

```bash
$ cat helloworld.kemono
```
